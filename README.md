# Titanic_Survival_Prediction

A Machine Learning project that predicts whether a passenger survived the Titanic disaster using classification algorithms.

## Project Overview
This project uses the Titanic dataset to train a model that predicts passenger survival based on features like age, gender, ticket class, and fare.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Used
- Logistic Regression

## Features
- Data preprocessing
- Handling missing values
- Model training and testing
- Accuracy evaluation
- Confusion matrix visualization
